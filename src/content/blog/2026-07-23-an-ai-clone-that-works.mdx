## AI can actually produce good results

I'm gonna prove it to you by showing you real examples of what the AI has created instead of just talking about what it can do.

For these examples, I tried drafting responses to three different Reddit posts. As you know, Reddit is incredibly punishing, so the responses have to be good.

And the responses came back way better than what I expected — or could have written myself — for the amount of energy and time spent.

## Second Example

Here's the second example. This person has been using Tailwind and React for a year, but they struggle with understanding CSS layouts.

<Image
  src="/src/assets/blog/an-ai-clone-that-works/reddit-post-learning-css-layouts.png"
  alt="A Reddit post from a developer who has used React and Tailwind for a year, asking whether to learn CSS fundamentals from the beginning or focus on layout concepts like the box model, flexbox, grid, and sizing behavior"
/>

Here's the response my AI created:

<Image
  src="/src/assets/blog/an-ai-clone-that-works/clone-response-learning-css-layouts.png"
  alt="The AI clone's response explaining that percentage heights resolve against the parent's height, and recommending the poster focus on the box model, how boxes decide their size, then flexbox and grid"
/>

This did not blow me away as much as the previous one.

But it holds some of my communication patterns, like:

- I will not tell anyone to start from scratch but instead learn what they need
- I also give people what to focus on if I'm in a good mood
- It contains some details of how I think about the box model — like whether margins or paddings should depend on the parent or the content

The nice thing here is that the second paragraph — about `height: 100%` — is something I have not written in any of my course materials.

I didn't write that because I don't see a point to educate about a wrong mental model. (You don't have to know things you don't need to know).

But the AI was able to pull out the information from general knowledge and write them in a way that flows with the rest of the message.

Overall, I'd give this a B+ grade if I were marking it for awesomeness.
